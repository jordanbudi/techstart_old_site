Building a World

Let’s learn how to add and manipulate objects in Vizor! Many Vizor techniques work in other CG programs too.

The Patches Menu

In Vizor, you add 3D objects using the “Patches Menu.” The Patches Menu contains two tabs.

  • The “Objects” tab shows objects you can add to your scene.
  • The “Patches” tab allows you to use programming in Vizor and will be explored in later projects.

If you do not see the Patches Menus, click the “Patches” button in the Vizor editor. You can click again to make the Patches Menu disappear. You can also click on the Patches Menu and drag to move the menu around.


Click the Patches button on the top right to turn the Patches menu on and off.


Remembet to hit “Tab” if your Editor ever looks like the following. We will cover Patches and programming in later projects.

Hello Objects!

Now let’s add an object to your world!

Do It Yourself!
  • Select the Objects tab in the Patches menu.
  • Select the objects “Objects: Basic” option, then select the “Box” option.

Click “Box” in the “Objects: Basic” option on the Patches Menu to add a Box.

You should now see a box object in your scene. You can select any object by clicking on the object. You know you’ve selected an object when red, green, and blue lines appear by the object. Each line is an “axis” and can be used to change the object’s position, size, and rotation.

Object Manipulation

The following videos show how you can manipulate objects. The “Move,” “Scale,” and “Rotate” buttons can be found on the rop right of the Vizor editor.


Select the “Move” button then drag along the axes to move an object.


Select the “Scale” button then drag along the axes to change an object’s size.


Select the “Rotate” button then drag along the axes to rotate an object.

Extra Tips
  • You can delete an object by selecting the object and pressing “Backspace” or “Delete” on your keyboard.
  • You can undo changes by pressing the “Ctrl” and “Z” button on your keyboard.
  • With an object selected, you can press “Alt” and move the object to create a copy.

Building a World

Now you’re all set to start building your first virtual world!

Do It Yourself!
  • Add at least 5 objects to your scene and modify the objects as much as you want. Try the different options in the Objects menu!
  • Make a scene you would be happy to show your friends.
  • Once you are ready, go to the next section to view and publish your project.
